# MartianEclipses
An educational tool to demonstrate how lunar eclipses occur by considering Martian eclipses of Phobos and Deimos.
Eclipse not implemented yet, but visibility done! Still need to implement dependency and all that stuff.

## How to run
After cloning the repo, run python main.py
